# AR Analiz Platformu (Satış Takibi Sistemi)

Bu proje, işletmelerin satış verilerini anlık olarak takip edip grafiksel raporlar üretebildiği, PHP ve MySQL tabanlı bir web uygulamasıdır. 

## Özellikler
* Satış verilerinin sisteme kaydedilmesi ve dinamik listeleme
* Chart.js kullanılarak bölge ve kategori bazlı grafiksel analizler
* Toplam ciro ve satış adetlerini gösteren yönetim paneli (Dashboard)
* Satış raporlarının PDF formatında dışa aktarılması
* Kullanıcı dostu arayüz ve Gece Modu (Dark Mode) desteği

## Kullanılan Teknolojiler
* Backend: PHP 8, PDO
* Frontend: HTML5, CSS3, Bootstrap 5, Chart.js
* Veritabanı: MySQL
* Sunucu Ortamı: XAMPP

## Kurulum
1. Proje dosyalarını XAMPP "htdocs" klasörünün içine taşıyın.
2. XAMPP üzerinden Apache ve MySQL servislerini başlatın.
3. phpMyAdmin üzerinden proje için yeni bir veritabanı oluşturun.
4. Proje dizinindeki SQL dosyasını oluşturduğunuz veritabanına içe aktarın.
5. Tarayıcınızdan http://localhost/proje_klasor_adi adresi üzerinden sistemi çalıştırın.
